Tests 1-7 are all valid and should generate the corresponding output.

Test 8 has min=max and hence should return -1.

For test 9, first create a file named: "output9", then change its file permissions to 000,
then run pa2 with test9 and output9, and it should return -1 because the output file can't be opened.

Running the makefile successfully has 10 points.

The 9 test cases have 10 points each.

The whitespaces (including the newline) in the outputs are not counted,
so you may have the header and all the pixel values in just one line, or arranged in columns and rows,
based on the width and height.

Comments in the outputs (if any) are ignored.
